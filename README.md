💱 Currency Converter Web App

A simple and responsive Currency Converter web application that allows users to convert one currency into another using real-time exchange rates.

🚀 Features
🌍 Convert between multiple international currencies
🔄 Real-time exchange rate fetching using API
🇺🇸 Dynamic country flags based on selected currency
📱 Responsive and clean UI
⚡ Default conversion on page load


🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling
JavaScript (Vanilla JS) – Logic & API handling
Currency API – Exchange rates
Flags API – Country flags


📂 Project Structure (Currency-Converter)

│── index.html      # Main HTML file
│── main.css        # Styling file
│── app.js          # Main logic & API calls
│── codes.js        # Currency-country mapping

⚙️ How It Works
User enters an amount
Selects "From" and "To" currencies
Clicks Get Exchange Rate
App fetches real-time data from API
Displays converted amount


🔗 API Used -> "https://cdn.jsdelivr.net/gh/fawazahmed0/currency-api@1/latest/currencies"
